// About

何を作っているかより
どんな人かが先に伝わる場所へ

鮫月ナツネのプロフィールは、実績の羅列よりも「何を大事にして、どんな空気を作る人なのか」を先に感じてもらう構成にしています。

鮫月ナツネは、VRChatを中心に活動する個人クリエイター。ワールド制作、ギミック開発、動画発信、イベント企画、コミュニティ運営まで、表現の形をひとつに絞らず、その時いちばん面白いと思う方法で人と場をつないでいくタイプです。

技術だけを見せるのではなく、そこに集まる人の体験や空気感まで含めてデザインすることを大切にしています。

Core Value

すごいものを作るだけでなく、人が入りやすく、居たくなり、また来たくなる空気を作ること。

Axis

作る、見せる、集めるを分けずに、ひとつの世界観として活動全体を育てていくこと。

Design

教材サイトではなく、人物の魅力と温度感が伝わる公式ポータルとして設計。

Target

VRChatユーザー、作品を見たい人、活動を応援したい人、コミュニティが好きな人。

Profile Note

作品ごとに人が違って見えるのではなく、どの活動を見ても「鮫月ナツネらしい」と感じられることを重視。

Keyword

ポップ / 親しみやすさ / 世界観 / 遊び心 / 技術と体験の両立

For Visitors

はじめて来た人には「何者か」がすぐ分かる。既に知っている人には「今なにしてるか」が追いやすい。
