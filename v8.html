// ABOUT

何を作っているかより
どんな人かが先に伝わる場所へ

実績の羅列よりも「何を大事にして、どんな空気を作る人なのか」を先に感じてもらう構成。

鮫月ナツネは、VRChatを中心に活動する個人クリエイター。ワールド制作、ギミック開発、動画発信、イベント企画、コミュニティ運営まで、表現の形をひとつに絞らず、その時いちばん面白いと思う方法で人と場をつないでいくタイプです。

技術だけを見せるのではなく、そこに集まる人の体験や空気感まで含めてデザインすることを大切にしています。作品単体ではなく、「この人の活動を追っていくと面白い」と思ってもらえる存在を目指しています。

CORE VALUE

人が入りやすく、居たくなり、また来たくなる空気を作ること。

AXIS

作る、見せる、集めるを分けずに、ひとつの世界観として育てていく。

DESIGN

人物の魅力と温度感が伝わる公式ポータルとして設計。

TARGET

VRChatユーザー、作品を見たい人、活動を応援したい人。
